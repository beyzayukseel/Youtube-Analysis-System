# Youtube-Analysis-and-Recommendation-System
The aim of our project is to create a useful product for people who will produce content on youtube. We have created a number of images, colors, links, and various graphics by category.
In this way, we enable a person who will produce content to choose a content from which category and which color tones they can use, and to easily find links that may be useful to him.
